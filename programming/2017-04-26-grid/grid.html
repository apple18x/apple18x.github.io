CSS Grid 入门

CSS Grid 是 CSS3 的一种新的布局方式，简单来说，就是一种网格布局的方式。

Grid vs Flex

CSS Grid 是二维的网格布局，而 CSS Flex 则是一维布局，至于选择哪个，在实际应用当中很好区分。目前 CSS Grid 已经在最新的主流浏览器上的到了较好的支持。

从 CSS Grid Garden 以小游戏的形式介绍了 Grid 相关的几个属性，很有意思。以下翻译了部分任务介绍，并展示了实现代码。

原文：http://cssgridgarden.com/

欢迎来到 Grid 花园，在这里，你可以使用 CSS 来种植胡萝卜，使用 grid-column-start 属性来给你的胡萝卜浇水吧。

例如，grid-column-start: 3; 代表将浇水区域从三列开始。

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-column-start: 3
}

啊，如果像下面这样，角落里有一块杂草，那么用 grid-column-start 应该写成什么呢？

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#poison {
  grid-column-start: 5;
}

如果你只用 grid-column-start 属性，那么只能选取单列但不能选取多列，添加 grid-column-end 属性就可以完美解决这个问题。 

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-column-start: 1;
  grid-column-end: 4;
}

很明显，我们选择了1～3列，但不包含第4列，即1 ≤ 选择列 < 4。

当我们成对使用 grid-column-start 和 grid-column-end 属性时，我们会以为grid-column-end 的值一定要大于 grid-column-start，事实并不是如此。
在下面这个例子里，grid-column-end 的值更小。

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-column-start: 5;
  grid-column-end: 2;
}

这里代表2 ≤ 选择列 < 5，究竟是否包含列数并不是由开始还是结束点决定的，而是由大小值决定的。

如果想从右侧开始计算，你可以给 grid-column-start 或 grid-column-end 设为负值。例如，grid-column-end: -1 代表从右开始数的第一列。

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-column-start: 1;
  grid-column-end: -2;
}

现在我们来试试 grid-column-start 设为负值。grid-column-start: -2 代表从右边数的第一列。

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#poison {
  grid-column-start: -3
}

除了用开始坐标和结束坐标的方式来选择网格，我们还可以用 span 这个关键字来表示选择的长度。在下面的例子中，代表起始位置为第二列，选择的长度为2。这种写法等同于：

#water {
  grid-column-start: 2;
  grid-column-end: 4;
}

特别需要注意的是，span 不能为负值哦。

另外你也可以在 grid-column-start 属性中使用 span 关键字来设置相对于结束位置的长度。

如果你觉得每次都要同时设置 grid-column-start 和 grid-column-end 很麻烦，可以用 grid-column 来简写，语法如下：

grid-column: <grid-column-start> / <grid-column-end>

例如：grid-column: 2 / 4 表示 grid-column-start 的值是2，grid-column-end 的值是4。

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-column: 4 / 6
}

在使用 grid-column 简写的方法里，也支持 span 关键词。

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-column: 2 / span 3;
}

Grid 与 Flex 布局方式最大的不同就是它可以从 columm 和 row 两个维度来进行定位。 grid-row-start 和 grid-column-start 的使用方法相同，但代表的是行而不再是列，就像下面这个例子一样。

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-row-start: 3
}

现在试试简写的 grid-row: grid-row-start / grid-row-end 语法。

还可以同时设置 grid-column 和 grid-row 两个维度的数值来选择某一个特定的网格。

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#poison {
  grid-column: 2/2;
  grid-row: 5/5
}

你也可以同时使用 grid-column 和 grid-row 来选择一个区域。

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-column: 2/6;
  grid-row: 1/6
}

如果你觉得同时写 grid-column 和 grid-row 太麻烦了，那么你可以试试 grid-area，语法如下：

grid-area: grid-row-start / grid-row-end / grid-column-start / grid-row-end

例如：grid-area: 1 / 1 / 3 / 6;

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-area: 1/2/4/6
}

以上考虑的都是单个区域的情况，如果选择了多个交叉区域会怎样呢？答案是完全没有问题。在下面的例子里，我们用 grid-area 来定义了第二个交叉区域。 

#garden {
  display: grid;
  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water-1 {
  grid-area: 1 / 4 / 6 / 5;
}

#water-2 {
  grid-area: 2/3/5/6
}

到目前为止，我们一直考虑的都是 5*5 网格情况，其中每个小格的宽是20%，高也是20%。这是由于设置了以下两个属性：

grid-template-columns: 20% 20% 20% 20% 20%; 
grid-template-rows: 20% 20% 20% 20% 20%;

每个属性包含5个值，对整体划分为5行5列，20%表示每一行（列）的宽（高）都是20%。当然你也可以设为自己需要的值，例如下面这样。

#garden {
  display: grid;

grid-template-columns: 50%
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-column: 1;
  grid-row: 1;
}

如果每一列都要设定一个具体的值可能会带来很多的不便，repeat 方法则会帮你摆脱这样的困境。例如，之前我们的写法都是 grid-template-columns: 20% 20% 20% 20% 20%;，我们也可以用 grid-template-columns: repeat(5, 20%);来取代。

#garden {
  display: grid;

grid-template-columns: repeat(8, 12.5%);
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-column: 1;
  grid-row: 1;
}

grid-template-columns 属性并不是只接受百分比，也接受长度单位，比如 pixel 和 em，你也可以混合使用它们。

在下面的例子里，我们分别设了三个不同单位的值，分别是 100px,3em和40%。

#garden {
  display: grid;

  grid-template-columns: 100px 3em 40%;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

在 CSS Grid 中有一个全新的单位 —— fr，1fr 代表可用空间中的一个单位。
例如两个元素的宽分别设为 1fr 和 3fr，那么整个宽度将会被分为4等分，第一个元素宽度为1/4，第二个元素宽为3／4。

#garden {
  display: grid;

  grid-template-columns: 1fr 5fr;
  grid-template-rows: 20% 20% 20% 20% 20%;
}

如果grid-template-columns的属性值中出现了不同的单位，其中包含pixels, percentages 或 em，那么如果同时又包含 fr，则 fr 是以剩余空间为基础的。

#garden {
  display: grid;

  grid-template-columns: 50px 2fr 2fr 1fr 50px
  grid-template-rows: 20% 20% 20% 20% 20%;
}

#water {
  grid-area: 1 / 1 / 6 / 2;
}

#poison {
  grid-area: 1 / 5 / 6 / 6;
}

下面我们再来看一个 px 和 fr 混合使用的例子。

#garden {
  display: grid;

  grid-template-columns: 75px 3fr 2fr
  grid-template-rows: 100%;
}

grid-template-rows 的使用方法和 grid-template-columns 完全一样。

grid-template-rows: <length> <percentage> <flex> max-content min-content minmax(min,max)

#garden {
  display: grid;

  grid-template-columns: 20% 20% 20% 20% 20%;
  grid-template-rows: 12.5px 12.5px 12.5px 12.5px 1fr;
}

#water {
  grid-column: 1 / 6;
  grid-row: 5 / 6;
}

grid-template 是 grid-template-rows 和 grid-template-columns 的简写形式。

grid-template: <grid-template-rows> / <grid-template-columns>

例如 grid-template: 50% 50% / 200px; 将会把行分为 50% 高的相等两行，列宽为 200px。

#garden {
  display: grid;

  grid-template: 60% 40% / 200px;
}

#water {
  grid-column: 1;
  grid-row: 1;
}

最后一个练习~

#garden {
  display: grid;
  grid-template: 1fr 50px / 1fr 4fr
}
