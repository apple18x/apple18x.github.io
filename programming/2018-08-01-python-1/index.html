  • 无法”看见“的字符：\n(换行)、\t、\r(回车)

问：如何输出（print）字符串中的 \n，而非被转义？
答：
方法一：print("hello\\nworld\\nworld")
方法二：print(r"hello\nworld\\nworld")
注：r 表示后面的字符串为原始字符串，而非普通字符串

元组(Tuple)

元组

注：元组与列表的区别

  1. 访问/截取元素：下标（同列表）

以上，字符串、列表和元组均称为序列。序列具有一些通用操作：

  1. 每个元素都被分配一个序号，可通过下标访问元素
  2. 切片：[start_pos: end_pos: step]
  3. 可加和
  4. 判断元素是否（不）存在序列中：not in
  5. 判断序列长度：len()
  6. 其他函数：max()，min()，ord() 等

集合(Set)

集合最重要的特性：

  1. 无序
  2. 不重复
  3. 与序列相同的方法：len()，in，not in 等
  4. 特殊方法：-(差集)，&(交集)，|(并集)

同时，不具有序列的部分特性：

  1. 通过下标
  2. 切片
  3. …

字典(Dict)

字典不是序列，属于一种集合，字典和集合的均用 {} 定义，用逗号分隔。
字典与集合的最大区别在于定义方式不同，集合只有 value 没有 key，而字典是通过 {key:value} 的形式来定义的。

注：value 可以为 python 中任意的类型，key 必须为不可变类型

相关操作：

  1. {key: value}[key]，通过 key 访问 value
