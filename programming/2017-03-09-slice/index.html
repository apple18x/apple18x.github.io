slice(0)

发表于 2017-03-09   |   分类于 技术

今天看代码时看到了 slice(0) 的写法，简单研究了一下。

JS 中对象（数组也是对象）是索引传递的，赋值时会被共享，而不是复制副本。所以只要一个对象被修改，被赋值的变量也会被改变。
