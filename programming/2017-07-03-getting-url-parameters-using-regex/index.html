正则表达式解析

在这个例子中，正则表达式为：

(1) 参数通常在开头(^)或以 & 进行连接
(^|&)

(2) 属性值设定
([^&]*)

[^] 代表匹配未在方括号中指定的字符。添加 &，这样，该正则表达式匹配任何非 & 字符。* 代表出现 0 次或多次，因此只要不出现 &，均被认为是属性值的一部分。

(3) 以 & 作为结束或遇到结尾 ($)
(&|$)
