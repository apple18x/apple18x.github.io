解法的优缺点及比较

  • 浮动方案

优点：兼容性好
缺点：脱离文档流，需要清除浮动，处理周边元素关系。可使用性比较差

  • 绝对定位方案

优点：快捷
缺点：脱离文档流

  • flexbox 方案

优点：解决了以上两种方案的问题
缺点：不兼容低版本浏览器

  • 表格方案

优点：兼容性好
缺点：三栏同时增高

  • grid 方案

优点：方便易用
缺点：兼容性

如果高度未知或内容超出，浮动方案、绝对定位方案和 grid 方案将不再适用。
