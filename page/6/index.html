HTTP 协议

发表于 2017-10-20   |   分类于 技术

1、HTTP 协议的主要特点

  • 简单快速
    uri 是固定的，访问简单
  • 灵活
    HTTP 头部表明了数据类型，通过一个 HTTP 协议有可以完成不同数据类型的传输
  • 无连接
    连接一次就断开，不会保持持久连接
  • 无状态
    客户端向服务端建立连接，完成后连接断开；下次客户端发送请求，服务端无法判断是否是同一个身份。即服务端是不记录状态的。
